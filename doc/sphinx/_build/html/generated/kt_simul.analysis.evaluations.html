kt_simul.analysis.evaluations¶

Evaluations are small plugin that can be automatically launch after a simulation.

kt_simul.analysis.evaluations.find_evaluations()¶

This function return a list of Evaluation classes that are enabled. In the future, it will have the capability to filter evaluations.

Note

Source from http://www.luckydonkey.com/2008/01/02/python-style-plugins-made-easy/

Return type:list
Returns:a list if classes that are subclasses of cls
class kt_simul.analysis.evaluations.Evaluation¶

Abstract class. All evaluation plugin need to write a class that inherit from Evaluation.

Warning

run() method need to be implemented.
